## v1.1.0

### 项目管理

- **新建项目向导** — 模态窗口引导创建，支持选择父级目录、文件夹名校验、模板选择（默认模板 10 张卡片 / 空模板 1 张卡片），可选择当前窗口打开或新窗口打开
- **项目模板** — 内置两种模板：默认模板（全部 10 张预设卡片），空模板（仅 1 张自定义卡片，从零开始）
- **文件打开前校验** — 打开项目前验证 `userData.json` 是否合法

### 导入数据

- **文件导入** — 从 `.md`/`.txt` 文件导入项目数据，自动识别任务边界（`##` / `【】`）和卡片字段（`**卡片名**：`），重名任务支持跳过/覆盖/重命名等多选项
- **拖拽导入** — 拖 `.md`/`.txt` 到侧边栏触发项目级导入，拖到工作区触发卡片级导入
- **粘贴导入（解析数据）** — 粘贴文本自动检测格式（任务级/卡片级），一键导入

### 导出数据

- **项目导出** — 导出全部任务为 Markdown / 纯文本，默认文件名 `{项目名}_{日期}_{时间戳}`
- **任务右键导出** — 右键任务项 → 导出当前任务为 `.md`
- **卡片右键导出** — 右键卡片 → 导出单张卡片为 `.md`
- **拖拽导出卡片** — 按住卡片主体拖出窗口，松手导出到桌面
- **拖拽导出任务** — 按住任务序号标记拖出窗口，松手导出到桌面

### 任务管理

- **右键菜单** — 任务项右键菜单：重命名 / 删除 / 复制 / 导出任务 / 上方插入空模板 / 上方插入默认模板 / 下方插入空模板 / 下方插入默认模板 / 重构帧位置
- **任务自动保存** — 切换任务、关闭项目、导出前、主进程请求关闭前自动保存

### 快捷键系统

- **23 个可配置操作** — 支持按键绑定/解绑、启用/禁用、单个恢复默认、全部恢复默认
- **覆盖范围** — 文件（保存/打开/新建/关闭/导入/导出）、任务（新建/删除/重命名/复制）、编辑（复制预览/清空/添加自定义卡片）、导航（任务切换/卡片切换/侧边栏）、工具（搜索/设置）、布局（重置）

### 全局搜索

- `Ctrl+Shift+F` 跨所有任务全文搜索卡片内容
- 结果按任务分组，点击跳转到对应卡片并高亮 2 秒

### 右键菜单优化（卡片）

- 卡片右键菜单：重命名 / 清空 / 删除（隐藏预设卡片/删除自定义卡片） / 导出
- 自定义卡片支持动态增删

### AI 规范文档

- 文件 → AI 规范，半模态窗口查看 AI 数据输出格式规范
- 顶部中英文切换，底部一键复制原始文本

### 主题

- 深色 / 浅色 / 默认 / 跟随系统 四种主题
- 切换实时生效

### 实例锁与进程管理

- **单实例锁** — `%APPDATA%\ai-prompt-helper\.app-lock.json`，防止多开
- **僵尸进程检测** — 锁超时 45 秒自动接管
- **看门狗** — 窗口 15 秒未显示则自动退出
- **`--force` 参数** — 强制终止旧进程并重新启动

### 日志系统

- **四级日志** — ERROR / WARN / INFO / DEBUG
- **17 个分类** — LIFECYCLE / FILE / TASK / UI / IPC / ERROR / LOCK / CONFIG / WINDOW / THEME / LANGUAGE / SETTINGS / SHORTCUT / PROJECT / IMPORT / EXPORT / SEARCH
- **7 天自动清理** — 每日中英文各一份（`yyyy-MM-dd_zh-CN.txt` / `yyyy-MM-dd_en.txt`）
- **调试预览** — 终端风格实时日志查看器，颜色标识日志级别

### 语言

- 中文 / English 切换，重启生效
- `--cn` 参数强制 Chromium 内置页面为中文

### 关闭行为

- 三种模式：退出应用 / 隐藏到托盘 / 隐藏到任务栏
- 托盘支持双击恢复、右键退出

### 最近项目

- 最近 20 个项目，可单个移除，路径失效灰色标记

### 界面优化

- 开始页面优化，未打开项目时显示引导入口
- 卡片布局支持拖拽排序、折叠展开、高度调整
- 状态栏显示保存状态、路径、任务数、卡片数
- 多窗口支持（文件 → 新建窗口）

### 性能与稳定性

- 模块拆分 — `projectImport.js`、`globalSearch.js`、`settingsDialog.js`、`createProjectDialog.js`、`shortcuts.js`、`aiSpecContent.js`、`mdToHtml.js` 等独立模块
- 首帧合成后显示窗口，避免白屏闪烁
- 关闭保存超时保护，防止卡死导致僵尸进程
- 防抖显示窗口，防止消息堆积

### 启动参数

- `--force` — 强制启动，跳过实例锁
- `--test` — 开启 Chrome DevTools 远程调试（9222）+ chrome://tracing
- `--cn` — Chromium 内置页面中文
- `--no-sandbox` — 关闭沙箱（开发模式默认）
