教学：如何编写标准输入文件

一、什么是标准输入文件

标准输入文件是一个 .md 文件，你可以用任意文本编辑器手动编写，然后通过菜单 文件 → 导入项目数据 导入到 AI_Helper，批量创建任务和卡片。

它的本质是一个结构化的 Markdown 文件：## 标记任务，**卡片名**： 标记卡片及内容。

二、基本格式规则

1. 任务 — 用 ## 标记

每个任务以 ## 开头，后可跟可选编号和任务名称。一个 ## 就是一个任务。

## 1. 赛博朋克城市
## 2. 森林精灵

2. 卡片 — 用 **卡片名**： 标记

粗体包裹卡片名称，紧跟冒号（： 或 :），冒号后同行写内容。卡片名必须与标准卡片名称完全一致。

**主体特征**：一位穿着黑色风衣的赛博格侦探。
**场景环境**：雨夜的霓虹街道，全息广告牌闪烁。
**光影色彩**：冷色调为主，高对比度。

3. 多行内容

不以 ## 或 ** 开头的行，自动作为上一张卡片的续行内容。

**场景环境**：雨夜的霓虹街道。
全息广告牌在林立的高楼间闪烁。
地面倒映着紫色和蓝色的光。

三、10 个标准卡片名称

#卡片名称描述
1主体特征画面主体的外貌、姿势、材质
2场景环境背景、地点、环境氛围
3光影色彩光源方向、色调、对比度
4艺术风格画风、流派、渲染风格
5镜头景别构图、景别、拍摄角度
6镜头运动摄影机运动方式
7时间节奏帧率、慢动作、延时
8动态事件画面中的动作、事件
9技术参数分辨率、画幅比、对焦
10负面排除不希望出现的元素

也可以用自定义卡片（如"声音设计"），名称自定，导入时自动创建。

四、完整示例

单个任务

## 1. 赛博朋克雨夜

**主体特征**：一位穿着黑色风衣的赛博格侦探，左眼是红色机械义眼，右臂露出银色金属骨骼。

**场景环境**：雨夜的霓虹街道，全息广告牌在林立的高楼间闪烁，地面倒映着紫色和蓝色的光。

**光影色彩**：冷色调为主，霓虹灯的红紫色光作为主光源，雨中反光营造迷离氛围。

**艺术风格**：赛博朋克风格，高对比度，电影级调色，Blade Runner 风格美学。

**镜头景别**：中景，半身构图，低角度仰拍。

**镜头运动**：从人物背后缓慢环绕至正面，手持微晃动。

**时间节奏**：慢动作，每秒 60 帧。

**动态事件**：人物点燃一支烟，烟雾在细雨中缓缓上升。

**技术参数**：4K，宽银幕 21:9，浅景深，电影级色彩分级。

**负面排除**：无阳光，无噪点，无模糊，无文字，无水印。

五、常见错误

  • 卡片名冒号后换行写内容 — 内容必须与冒号在同一行
  • 卡片名拼写错误 — 如"光影氛围"应为"光影色彩"
  • 误用 ### 三级标题 — 那是卡片导入格式（工作台拖拽），不是项目导入格式
  • 误用 **内容** 独占一行 — 那也是卡片导入格式
  • 添加 # 一级标题 或元信息行 — 解析器可能误解析

六、如何导入

  1. 用任意文本编辑器编写 .md 或 .txt 文件
  2. 在 AI_Helper 菜单栏点击 文件 → 导入项目数据
  3. 选择文件即可导入

七、让 AI 帮你生成

将菜单栏 AI规范 中的规范内容复制为 System Prompt 发给 ChatGPT / Claude 等 AI，AI 就能按正确格式生成 .md 文件，你保存后导入即可。

1. What is a Standard Input File

A standard input file is a .md file that you write with any text editor, then import via File → Import Project Data to batch-create tasks and cards in AI_Helper.

It is structured Markdown: ## marks tasks, **CardName**：content marks cards.

2. Format Rules

1. Task — marked by ##

Each task starts with ## , followed by an optional number and the task name.

## 1. Cyberpunk City
## 2. Forest Elf

2. Card — marked by **CardName**：

Card name in bold, followed by a colon (： or :), with content on the same line. Card names must match the standard names exactly (in Chinese).

**主体特征**：A cyborg detective in a black trench coat.
**场景环境**：Neon-lit streets on a rainy night.
**光影色彩**：Cool tones, high contrast.

3. Multi-line content

Lines not starting with ## or ** are appended to the previous card as continuation.

**场景环境**：Neon-lit streets.
Holographic billboards flicker among skyscrapers.
Ground reflects purple and blue light.

3. Standard Card Names (10 Dimensions)

#Card NameDescription
1主体特征Subject — appearance, pose, material
2场景环境Scene — background, location, atmosphere
3光影色彩Lighting — light direction, tone, contrast
4艺术风格Style — art style, genre, rendering
5镜头景别Shot Scale — composition, shot size, angle
6镜头运动Camera Movement — camera motion
7时间节奏Time & Rhythm — frame rate, slow motion
8动态事件Action — events, changes in frame
9技术参数Technical — resolution, aspect ratio, focus
10负面排除Negative — elements to exclude

Custom cards (e.g. "Sound Design") are also supported — auto-created on import.

4. Examples

Single Task

## 1. Cyberpunk Rainy Night

**主体特征**：A cyborg detective in a black trench coat, left eye replaced with a red mechanical prosthetic, right arm exposing silver metal skeleton.

**场景环境**：Neon-lit streets on a rainy night, holographic billboards flickering among skyscrapers, ground reflecting purple and blue light.

**光影色彩**：Predominantly cool tones, red-purple neon light as the main light source, rain reflections creating a hazy atmosphere.

**艺术风格**：Cyberpunk style, high contrast, cinematic color grading, Blade Runner aesthetic.

**镜头景别**：Medium shot, half-body composition, low-angle shot.

**镜头运动**：Slow orbit from behind the character to the front, subtle handheld shake.

**时间节奏**：Slow motion, 60 fps.

**动态事件**：The character lights a cigarette, smoke rising through drizzle.

**技术参数**：4K, widescreen 21:9, shallow depth of field, cinematic color grading.

**负面排除**：No sunlight, no noise, no blur, no text, no watermark.

5. Common Mistakes

  • Putting content on a new line after the colon — content must be on the same line as **card**：
  • Misspelling card names — names must match the Chinese standard exactly
  • Using ### headings — that's the card-level import format, not project import
  • Using **content** on its own line — also the card-level format
  • Adding # headings or metadata lines

6. How to Import

  1. Write your .md or .txt file in any text editor
  2. In AI_Helper, click File → Import Project Data
  3. Select your file to import

7. Let AI Generate for You

Copy the spec from AI Spec in the menu bar as a System Prompt for ChatGPT / Claude. The AI will output correctly formatted .md files — save and import!
